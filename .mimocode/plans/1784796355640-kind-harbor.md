# Disable GradualBlur on Mobile & Tablet

## Task
Disable `GradualBlur` component on mobile (< 640px) and tablet (640px–1023px). It renders 5 stacked `backdrop-filter: blur()` layers that are GPU-heavy on smaller devices.

## Approach

### 1. Create `components/GradualBlurWrapper.tsx`
A client component that checks `window.innerWidth >= 1024` and conditionally renders `GradualBlur`. Returns `null` when viewport < 1024px.

- Uses `useState` + `useEffect` with resize listener
- SSR-safe: defaults to `true` (renders on server), updates on client mount

### 2. Update `app/layout.tsx`
- Replace `import GradualBlur from "@/components/GradualBlur"` → `import GradualBlurWrapper from "@/components/GradualBlurWrapper"`
- Replace `<GradualBlur ... />` → `<GradualBlurWrapper ... />`
- No other changes to layout (metadata stays as server component)

### 3. Add tests

**`sections/__tests__/layout.mobile.test.tsx`** (width=375):
- Test: GradualBlur is NOT rendered (no `.gradual-blur` element in DOM)

**`sections/__tests__/layout.tablet.test.tsx`** (width=768):
- Test: GradualBlur is NOT rendered

**`sections/__tests__/layout.xlarge.test.tsx`** (width=1920):
- Test: GradualBlur IS rendered (has `.gradual-blur` element)
